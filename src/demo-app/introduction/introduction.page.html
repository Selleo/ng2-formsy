Introduction

Forms are probably the most crucial aspect of your web application. Angular 2 forms markup can get quite verbose, particularly when displaying validation errors. Formsy provide UI components that simplify forms markup.

Formsy components wrap standard inputs and try to leave you as much flexibility in building forms markup as possible. We have started this project with both simple and advanced forms in mind.

Project was started on AngularAttack 2016 where we have been able to implement only few planned features. We hope to add more features and examples in future, like Material Design input container and inline editing container.

Features

Bootstrap field container

Wrap label, input and error messages in easy to use and flexible bs-input-container component

Validations

Display validations with bs-messages component (currently tied to ng2-translate).

Bootstrap horizontal form

Allow to control label size on form level, ensure errors are displayed right under field.

Install

npm i --save ng2-formsy

Usage

import {FORMSY_BS_DIRECTIVES} from 'ng2-formsy';
